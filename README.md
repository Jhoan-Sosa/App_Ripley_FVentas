# 🏦 Banco Ripley - Ecosistema Financiero

## 📋 Descripción
Sistema completo de originación de crédito para Banco Ripley, desarrollado como parte del curso de Desarrollo de Aplicaciones Móviles.

## 🛠️ Tecnologías
- **App Fuerza de Ventas**: Flutter (Web) - Para asesores de campo
- **Frontend React**: React + Vite - Panel de administración para supervisores
- **Backend**: FastAPI (Python) + Firestore - API REST
